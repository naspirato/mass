PyOD
Очень простой интерфейс: достаточно скормить numpy/pandas-данные и вызвать fit/predict.​

Автоматический выбор параметров, поддержка десятков моделей, большинство работают без серьезных ручных твиков.

Отлично подходит для табличных данных и простых временных рядов.

Пример:

python
from pyod.models.iforest import IForest
model = IForest()
model.fit(X)
anomalies = model.predict(X)  # 1 = outlier
alibi-detect
Ориентирован на “out-of-the-box” работу для большинства задач: outlier, drift detection и т.п.​

Готовые пресеты и автоматическая обработка метаданных, простая интеграция с массивом данных.

Может применять к числовым, временным и даже текстовым данным.

dtaianomaly
Специализация: временные ряды, требуется буквально несколько строк кода для запуска.​

Абстрактный API: fit/predict для большинства моделей.

Особенно хорош для мониторинга и долгих серий.

Scikit-learn (для простых случаев)
Если уже используешь sklearn — IsolationForest, LocalOutlierFactor требуют минимальных параметров.​
